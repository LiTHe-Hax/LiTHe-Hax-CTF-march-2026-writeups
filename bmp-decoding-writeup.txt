Looking at what is contained within the file header portion of BMP files reveals that there is a portion of the header that is not used and is usually 0.

Looking at the same portion in the BMP file you recieved, you will notice that it is not empty and actually spells out "h1d3". In other words, it spells out "hide" using letters and numbers.

This points toward the fact that you need to use the program "steghide" to extract the flag. If you then use steghide with "h1d3" as the password, you will get a text file that contains the flag.
